# Ripple Robot
A Velox-style amphibious robot

### Overview
This repository currently contains the CAD files for Ripple, a small amphibious robot inspired by fin-propelled Velox platforms. The design emphasizes simple manufacturing, modular components, and compatibility with waterproof servos and laser-cut fins.
